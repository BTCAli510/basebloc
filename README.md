# BaseBloc

Culture-led IRL→onchain onboarding on Base.

BaseBloc is an event command center and onboarding flow designed to turn real-world community participation into measurable onchain actions on Base. The product is being built to support culturally rooted activations, wallet-connected onboarding, RSVP and check-in flows, and attestations that create a bridge between in-person participation and long-term onchain engagement.

## What it does

BaseBloc is designed to help organizers:

- onboard people from IRL events into Base-native flows
- guide first-time users through wallet connection and participation
- issue participation-based attestations
- create repeatable onboarding loops tied to real community activity
- measure conversion from attendance to onchain action

## Current status

This project is currently MVP / pilot-ready.

Today, the app is focused on:
- wallet connection
- RSVP / event flow
- Base-native onboarding
- EAS-based participation logic
- preparing for live pilot activations

## Why it exists

Most onboarding into crypto is still abstract, speculative, or disconnected from real trust networks.

BaseBloc takes a different approach: start with real community, real events, and real participation, then route that energy into simple onchain actions on Base. The goal is not just acquisition, but repeat participation and long-term empowerment.

## Stack

Built on a Base-native stack, including:

- Next.js
- TypeScript
- OnchainKit
- wagmi
- MiniKit
- EAS
